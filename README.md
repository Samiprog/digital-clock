# ⏰ Digital Clock

A simple **React + Vite + Bootstrap** application that displays the current time in a digital clock format.

---

## ✨ Features

- Real-time digital clock  
- Updates automatically every second  
- Responsive design with **Bootstrap**  

---

## 🛠 Technologies

- React  
- Vite  
- JavaScript (ES6+)  
- Bootstrap 5
